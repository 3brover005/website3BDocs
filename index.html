Introducing 3B Documents & Electronic Signatures

An application built natively on the Salesforce platform for generating electronic documents and collecting signatures from Salesforce and non-Salesforce alike. The application is created using cutting edge technologies that will enable your business to scale without the added cost per transaction. Streamline agreement preparation, redlining, collaboration, and approvals. Safely and centrally store agreements for easy access and discovery.

Some of the main features the application offers are:

  • Nested Related Children
    Support rendering repeatable blocks with multiple nested child related objects
  • Unlimited Parent Relationships
    Add merge tags which traverse an unlimited number of parent relationships
  • Document Version Control
    Create a snapshot of a document each time a change is registered. Rewind back to an older version if needed
  • Reusable Sub-Templates
    Mark templates as "Sub-Templates" to allow embedding them in other templates. Support both Static and Dynamic sub-templates
  • Coming Soon! Contract Redlining
    Allow one or multiple collaborators to agree the terms of a contract. Assign permissions and approval process
  • Coming Soon! Assign Multiple Signatories
    Add multiple signatories to a document that requires an electronic signature. Add an order of signing and control the workflow
  • Embedded Input Elements
    Add HTML Elements like drop downs, checkbox(es), radio-buttons, text input, date input etc. to a template to collect user inputs
  • Extend Functionality With Code Snippets
    Using Custom Function Expressions, you can extend the application and integrate with charting software and even create your own logic for rendering
